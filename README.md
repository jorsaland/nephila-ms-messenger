# Nephila &#128376; - MS Messenger (0.0)

## Commands

### Start and stop service

#### Linux:

```
./scripts/start.sh [docker] [debug]
```

```
./scripts/stop.sh [docker] [debug]
```

#### Windows:

```
.\scripts\start.ps1 [docker] [debug]
```

```
.\scripts\stop.ps1 [docker] [debug]
```
